# c-sharp_Utils
* Some c# utils
* PS: Sorry for my english
* Nuget package: https://www.nuget.org/packages/CSharp.Mix.Utils/1.5.0

## Instalation
1) Go to releases
2) Find latest version
3) Download the dll
4) Add reference to it in your c# project
5) Add "using Utils;" at the top of your .cs file
6) Start using it
